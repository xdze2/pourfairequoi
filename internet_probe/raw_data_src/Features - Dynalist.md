## Feature highlights

Dynalist is featureful yet simple.

Without bloating the interface, tools show up only when you need them. See for yourself in the [demo](https://dynalist.io/demo); all panes can be closed for focusing on outlining and organizing.

So don't let the features scare you! They are all here to help you do your best work.

[See a full list of features](https://dynalist.io/features/full)
